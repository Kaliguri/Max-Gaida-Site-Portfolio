# Проекты

## Рекомендуемый порядок показа на главной (черновик Никси)

Под позиционирование «технический геймдизайнер, в первую очередь разработчик». Витрина —
**курируемая** (не все проекты подряд), от максимальной достоверности к технической глубине.
Финально сверим на живой вёрстке с реальными визуалами.

1. **Knock on the Coffin Lid** — единственный коммерческий зарелизенный тайтл (Steam + DLC, реальная
   студия RedBoon, 1 г 3 мес). Максимальная credibility; роль «технический ГД» — как раз мост.
2. **Bloodlines UI** — собственный зарелизенный продукт (Asset Store + itch.io, бренд Alebardium).
   Доказывает: довожу продукты до релиза сам, dev + UI/UX.
3. **Few Seconds, Many Deaths** — лид команды из 10, Steam, Steamworks API, живые метрики (~77k
   показов). Лидерство + техническая широта + реальная международная аудитория.
4. **Мультиплеер в R-Studio** — P2P, лобби, матчмейкинг (Photon / NGO). Самая «мясная» техническая
   глубина: нетворкинг хорошо продаёт middle-разработчика.
5. **METALHEAD** — 1 место на геймджеме, лид. Бейдж «делаю быстро и побеждаю».

Вторая полка / «ещё проекты»: Guildmaster (в работе — витрина паттернов), Tint Exporter (Figma-плагин,
широта инструментов), The Silent Eclipse (UE5 + красивое промо-видео, визуальное разнообразие),
MosPolyJam-3, TYPE-FASTER-MP (соло-джем).

Логика: первый экран — «зарелизил коммерцию и свои продукты», затем «вёл команды», затем «вот код и
нетворкинг». GD-грань вплетена через Knock и роль, а не вынесена отдельной воронкой.

## Учебные / Проектная деятельность (Московский Политех)

- **Few Seconds, Many Deaths** — **учебный** проект (ПД, Московский Политех) со страницей в Steam.
  Макс — **лид и инициатор ПД**, команда из 10 человек: организация работы, главный по тех. части +
  проджект-менеджер + лид. На этом проекте отлично освоил **Steamworks API**. Отличные отзывы
  комиссии; один из участников защитил свою часть на «отлично» (диплом).
  - https://github.com/Kaliguri/FewSeconds-ManyDeaths-Unity
  - https://store.steampowered.com/app/3259720/Few_Seconds__Many_Deaths/
  - Статистика Steam (скрины от Макса): ~77 059 показов, ~27 833 посещения страницы, коэффициент
    переходов 36.1%, ~441 в списках желаемого. География посещений: США 32%, РФ 26%, Франция 13%,
    Гонконг 6%, далее Китай / Аргентина / Бразилия / Сингапур / Канада / Германия — реальная
    международная аудитория. Сильный пруф для учебного релиза, можно показать как достижение.
- **The Silent Eclipse** — красивая игра на UE5, ПД в другом году, в команде. Есть классное промо-видео.
- **MVP для Яндекс.Игр и Android** — в составе команды из 4 человек (лид, художник, GD, Макс —
  основной программист):
  - PvP-стратегия в духе Clash Royale, но со строительством зданий в стиле тетриса.
  - Небольшая idle-RPG с кооперативом.
