# Опыт работы (студии)

Суммарно по PDF-резюме: ~1 год 5 месяцев формального опыта в студиях.

## RedBoon — Технический геймдизайнер (Unity)

- Подача роли: **технический геймдизайнер** (мост «разработчик + GD» — Макс одновременно и тот, и
  другой). В PDF значился как Game Designer; на сайте подаём как Technical Game Designer.
- Период: ноябрь 2023 — январь 2025 (1 год 3 месяца).
- Студия, разрабатывающая rogue-like игру **Knock on the Coffin Lid** (есть Steam, DLC).
  Этот же проект Макс отдельно называет флагманом — см. [`projects.md`](projects.md).
- Задачи:
  - Создание и поддержка нового контента (существа, карты, ивенты) на функционале ScriptableObject,
    реворк существующего.
  - Блок-схемы работы ивентов и их влияния на другие ивенты.
  - Работа с баг-репортами игроков и тестировщиков, исправление ошибок, изменение баланса.
  - Совместная работа с художниками, программистами, аниматорами.
- Стек: Unity, Git, GitLab, Fork, YouTrack, Google Docs, Figma.
- Результаты:
  - Глобальный реворк всех ивентов в игре.
  - Новая локация с уникальной механикой для DLC **«Nightmares of Millenis»**.
  - Новые достижения для Steam.
  - Комплексные модификаторы для забега в DLC, новые предметы, разговор на костре с союзниками,
    короткая версия ивентов для уникального игрового режима.

## R-Studio — Unity Developer (C#)

- Период: апрель 2025 — май 2025 (2 месяца).
- Студия, разрабатывающая игры под Android и Яндекс.Игры.
- Задачи:
  - Создание MVP проектов и их дальнейшая поддержка.
  - Изучение игр-конкурентов и реализация механик из них.
  - Создание Core и мета-геймплея.
  - Работа с UI/UX, VFX, анимациями и локализацией игры.
  - Создание билдов под требования платформ (например, Яндекс.Игры).
  - Поддержка архитектуры на ScriptableObject и ивентовой системы через EventBus.
  - Работа над оптимизацией, реализация Object Pool.
- Стек: Unity, C#, ООП, Git, GitHub, Fork, SOLID, EventBus, ECS, Android, WebGL, Яндекс.Игры,
  Plugin Your Games 2.0, Localization (Unity), Object Pool, Multiplayer, NGO, Photon, Odin Inspector,
  A* Pathfinding Project Pro.
- Результаты:
  - Мультиплеер (P2P) для нескольких игр: система лобби, матчмейкинг, рабочий геймплей для 1x1 и кооп.
  - Внедрил рекламу от Яндекс.Игр и метрики от Яндекс.Метрики.
  - Несколько MVP на основе игр разных жанров: стратегии, автобаттлеры, idle-RPG, tower defense.
