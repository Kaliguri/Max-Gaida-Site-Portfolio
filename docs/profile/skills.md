## Game programming (из стека R-Studio и опыта)

- ООП на C#, SOLID, паттерны.
- Архитектура: ScriptableObject-based, EventBus, ECS, Object Pool.
- Мультиплеер: NGO (Netcode for GameObjects), Photon, P2P, лобби, матчмейкинг (1x1 и кооп).
- Оптимизация, работа с физикой напрямую.
- Инструменты/пакеты: Odin Inspector, A* Pathfinding Project Pro, DOTween, Plugin Your Games 2.0,
  Localization (Unity).
- Монетизация/аналитика: реклама Яндекс.Игр, Яндекс.Метрика.
